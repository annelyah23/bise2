BASHROCK : SIMPLE ENCRYPT FOR BASH

Bashrock in tool for Encrypt bash script/shell but you can use bash script/shell as usual.

need:<br>
```
sudo apt install lolcat
```
usage:
```
bash bashrock.sh
```
